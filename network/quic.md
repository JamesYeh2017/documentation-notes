- TCP, Transmission Control Protocol   : 傳輸控制通訊協定
- UDP, User Datagram Protocol          : 使用者封包通訊協定
- QUIC, Quick UDP Internet Connections : 快速 UDP 網路連結

HTTP/QUIC 確定更名為 `HTTP/3`, 而此並非 `HTTP/1.1` 及 `HTTP/2` 的進版

# bash - issue

## bash進站歡迎訊息 (沒啥鳥用, 自己看的爽XD)

```sh
$ cat /etc/issue
\S
Kernel \r on an \m
```

script | Description
------ | --------------
\S     | OS name
\r     | OS edition
\m     | hardware grade(i386/i686...)
\t     | local time
\d     | local date
\l     | which tty
\n     | host name
\o     | domain name

# 目錄與檔案權限
- 2018/06/09
1. 檔案權限
2. 目錄權限


# 1. 檔案權限

> 檔案是實際含有資料的地方

- r: 可讀取檔案
- w: 編輯, 新增, 修改 檔案的內容(不含刪除)
- x: 檔案可被執行

僅針對「檔案的內容」, 與檔案名稱的存在與否無關


# 2. 目錄權限

> 目錄主要的內容, 在記錄檔案清單

- r: 具備讀取目錄清單的權限, ex: 使用 ls 查看
- w: 可以異動該目錄結購清單的權限(與檔名有關的異動啦)
    - 建立新的 檔案 && 目錄
    - 刪除已經存在的 檔案 && 目錄(不論該檔案的權限為何)
    - 移動檔案位置(更改名字)

- x: 目錄具備 x 權限, 表示使用者可以 cd 進入
